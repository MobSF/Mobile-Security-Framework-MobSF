# Login Customization and Branding (35-38)

- Branding variables (`MOBSF_BRAND_*`) control title, colours, and logo.
- Theme packs stored in `mobsf/static/themes/`; administrators can upload custom themes via admin UI.
- Authentication providers: local, OAuth2/OIDC, SAML, corporate SSO.
- MFA methods: TOTP, WebAuthn, backup codes.
- Failed login monitoring triggers alerts after configurable thresholds.
