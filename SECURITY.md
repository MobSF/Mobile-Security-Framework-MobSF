# Security Policy

## Reporting a Vulnerability (23)
- Email security@befly.com with details and proof-of-concept.
- Encrypt sensitive information using the PGP key fingerprint `A1B2 C3D4 E5F6` (public key in `docs/security/keys/security.asc`).
- Expected timeline: acknowledgement within 24 hours, update within 72 hours, fix ETA based on severity.

## Supported Versions
- Latest stable release (`main` branch) receives security patches.
- Previous minor release maintained for 90 days for extended support customers.

## Coordinated Disclosure
- We collaborate with reporters to coordinate public disclosure through release notes and CVE publication.
