About Mobile Security Framework

关于移动安全框架

移动安全框架（MobSF）是一个针对 Android、iOS 和 Windows Mobile 应用的安全研究平台。MobSF 可用于多种场景，如移动应用安全、渗透测试、恶意软件分析和隐私分析。静态分析器支持 APK、IPA、APPX 等主流移动应用二进制文件和源代码。动态分析器支持 Android 和 iOS 应用，提供交互式插桩测试、运行时数据和网络流量分析平台。MobSF 可通过 REST API 和 CLI 工具无缝集成到 DevSecOps 或 CI/CD 流水线中，轻松增强您的安全工作流程。
