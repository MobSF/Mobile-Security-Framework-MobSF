About Cyberspect

Cyberspect uses the open-source Mobile Security Framework (MobSF): an automated, all-in-one mobile application security assessment framework. MobSF supports mobile app binaries (APK, XAPK, IPA & APPX) along with zipped source code and provides REST APIs for seamless integration with your CI/CD or DevSecOps pipeline.
