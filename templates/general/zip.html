ساختار صحیح فایل ZIP

فقط ساختار های ios و android ( که با eclipse and Android Studio طراحی شده است) قابل قبول است .

پروژهای Eclipse:
فایل AndroidManifest.xml و شاخه 'src' باید در دایرکتوری ریشه فایلهای ZIPPED Source برای پروژه های eclipse وجود داشته باشد.

پروژهای Android Studio :
AndroidManifest.xml باید در 'app / src / main / AndroidManifest.xml' و شاخه "app / src / main / java /" باید برای پروژه های Android Studio وجود داشته باشد.

پروژهای iOS:
برای پروژه های iOS، فایل .xcodeproj باید در پوشه ریشه فایل های منبع ZIPPED وجود داشته باشد.
